# SAP Order Management Application (ABAP)

## Opis projektu
Aplikacja do rejestracji zamówień w systemie SAP.

## Funkcjonalności
- tworzenie zamówień
- wybór artykułów
- opisy produktów
- ilości
- generowanie numeru zamówienia przez SNRO
- zapis danych do tabel custom
- walidacja danych
- function modules

## Technologie
- ABAP
- Open SQL
- SAP Data Dictionary
- ALV

## Projekt 2
Raport ALV do podglądu zamówień
